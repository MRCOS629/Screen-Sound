# Screen-Sound
É um programa onde você pode dar avaliações e ver avaliações a uma banda. Foi desenvolvido em **C#** utilizando **.NET** no **Visual Studio**, com foco em praticar conceitos fundamentais de programação orientada a objetos, coleções e interação via console.

---

Algumas funcionalidades são:

Adicionar novas bandas  
Listar bandas cadastradas  
Avaliar bandas com notas numéricas  
Exibir a média de avaliações de cada banda  
Interface simples via console  

---

# 🛠️ Tecnologias usadas

- C#
- .NET
- Visual Studio
- Programação Orientada a Objetos (POO)

---

## Pré-requisitos

Antes de começar, você precisa ter instalado:

- [.NET SDK](https://dotnet.microsoft.com/download)
- [Visual Studio](https://visualstudio.microsoft.com/pt-br/) ou Visual Studio Code
